⚠️ ML Kit Limitations for Web

ML Kit is primarily for native mobile apps, not web browsers.

For web scanning, these work better:

  • Native Barcode API - Built into Chrome/Edge
  • ZXing API - JavaScript library
  • Html5-QRCode - Web-optimized

ML Kit Web Options:

1. Firebase ML Kit (Deprecated) - No longer supported

2. Cloud Vision API - Requires server + API keys

3. WebAssembly ML Kit - Complex setup, large files

Recommendation: Use the Native Barcode API scanner for best performance on supported browsers.
